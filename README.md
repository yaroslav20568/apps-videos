# Видео-демонстрации приложений

- **physics_ball.mp4** - Демонстрация приложения Physics Ball

  Это мобильное приложение на Flutter для симуляции физики мяча с гравитацией, отскоками и трением. Приложение позволяет пользователям наблюдать за падением мяча с гравитацией, его отскоками от стен с затуханием и трением при контакте с нижней границей экрана.

- **sky_flyer.mp4** - Демонстрация приложения Sky Flyer

  Это простая аркадная игра, в которой игрок управляет самолетом, нажимая на экран, чтобы поднять его вверх. Цель - пролететь как можно дальше, избегая столкновений с препятствиями и набирая очки.

- **animal_kingdom.mp4** - Демонстрация приложения Animal Kingdom

  Это мобильное приложение на Flutter, оформленное как интерактивная книга-энциклопедия о животных. Приложение позволяет пользователям открыть виртуальную книгу, листать страницы и изучать информацию о различных животных, включая их описание, среду обитания, рацион питания и продолжительность жизни.

- **quiz_science.mp4** - Демонстрация приложения Quiz Science

  Это мобильное приложение на Flutter для прохождения интерактивных квизов по трем предметам: математика, физика и химия. Приложение позволяет пользователям выбрать предмет, ответить на вопросы с множественным выбором и получить результаты с детальным разбором ответов.

- **todo_list.mp4** - Демонстрация приложения Todo List

  Это мобильное приложение на Flutter для управления задачами. Приложение позволяет пользователям создавать задачи, отслеживать их статус (ожидание, в процессе, выполнено), планировать задачи по дате и времени, а также сохранять все данные локально на устройстве.

- **time_tracker.mp4** - Демонстрация приложения Time Tracker

  Это мобильное приложение на Flutter для отслеживания времени работы над различными задачами. Приложение позволяет пользователям запускать таймер для задач, ставить на паузу, возобновлять отслеживание и автоматически сохранять накопленное время.
